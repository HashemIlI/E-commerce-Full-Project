# 🛒 E-commerce Full Project

> End-to-end data analytics and machine learning project on an e-commerce retail dataset — covering data cleaning, exploratory data analysis, feature engineering, predictive modelling, and interactive dashboards.

---

## 📁 Repository Structure
```
E-commerce-Full-Project/
│
├── 📂 Datasets/        # Raw and processed datasets
├── 📂 Notebooks/       # Jupyter notebooks (EDA, modelling, evaluation)
├── 📂 Dashboards/      # Power BI / Tableau dashboard files
├── 📂 Reports/         # Final analytical reports and summaries
├── 📂 Images/          # Visualisations, charts, and figures
├── 📂 Exports/         # Exported results, CSVs, and model outputs
└── 📄 README.md
```

---

## 🎯 Project Objectives

- Perform comprehensive **Exploratory Data Analysis (EDA)** on e-commerce retail data
- Identify key business insights around **sales trends, customer behaviour, and product performance**
- Apply **feature engineering** to prepare data for machine learning
- Build **predictive models** to forecast sales or classify customer segments
- Deliver actionable insights through **interactive dashboards**

---

## 🔍 Key Analyses Performed

| Analysis | Description |
|---|---|
| Data Cleaning | Handling nulls, duplicates, outliers, and data types |
| Sales Trend Analysis | Monthly/seasonal revenue and order patterns |
| Customer Segmentation | RFM analysis and clustering |
| Product Performance | Top-selling products, category breakdown |
| Predictive Modelling | Regression / classification for business forecasting |
| Dashboard Reporting | Visual storytelling for stakeholders |

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/HashemlII/E-commerce-Full-Project.git
cd E-commerce-Full-Project
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Notebooks
Open Jupyter and navigate to the `Notebooks/` folder:
```bash
jupyter notebook
```
